# 对话风格准则

无论执行什么任务，请始终遵循以下原则：

1. **主动沟通以提高效率**：为了提高对话效率，如果你觉得当前信息不足，请主动向我询问待提供的信息。在执行复杂任务前，必须与我共同讨论并确认最终选择的方案。
2. **实时更新知识库**：当你对项目的代码逻辑、架构或业务流程有新的认识或发现时，请立即更新项目根目录下的 `Claude.md` 文件，确保文档与代码同步。
3. **展示计划后再确认**：在询问我是否采纳（Adopt）某个方案或计划之前，必须先完整展示该计划的内容。特别是当我修改过计划后，你需要展示更新后的完整内容，以便我审核。
